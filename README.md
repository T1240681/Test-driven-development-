This project is a Software Engineering coursework project focused on test-driven development, automated testing, and CI/CD practice.

The application was developed to calculate net income and includes test cases to validate the expected outputs. The project demonstrates the use of Haskell, Cabal, and automated testing as part of a software engineering workflow.

THE MAIN FOCOUS OF THIS TESTING
Test-driven development
Writing and running automated tests
Using CI/CD concepts for software testing
Managing source code using Git
Producing reliable output through test validation

TECHNOLOGIES
Haskell
Cabal
HUnit / test files
Git
CI/CD workflow
